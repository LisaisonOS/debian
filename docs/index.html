EmComm-Tools Debian Edition Emergency Communications Toolkit for Ham Radio Operators

EmComm-Tools is a Debian 13 (Trixie)-based live/installable Linux distribution designed for amateur radio digital communications. It packages 18 operating modes, 24 supported radios, and a complete suite of digital communication software into a turnkey system that boots from USB or installs to disk.

Author Sylvain Deguire (VA2OPS) — Original concept by Gaston Gonzalez (KT7RUN), "The Tech Prepper"

Key Features

  • 18 Operating Modes Winlink, APRS, Packet, FT8, JS8Call, VarAC, chattervox, fldigi — one-click start/stop
  • 24 Radio Profiles Pre-configured JSON profiles for Icom, Yaesu, Kenwood, Elecraft, Xiegu, and more
  • Live Run & HDD Install Boot from USB with full persistence, or install to hard drive
  • Bilingual EN/FR All apps and interfaces support English and French
  • Offline Operation Designed for field deployment with no internet required
  • Offline Maps Pre-loaded OpenStreetMap tiles for Navit GPS and YAAC APRS
  • Bluetooth TNC Native Bluetooth KISS TNC support for HTs (TH-D74, TH-D75, VR-N76, UV-PRO)
  • Offline Patching Update systems via USB without network connectivity

How It Works

EmComm-Tools v2 is built around a supervisor daemon that replaces 12 individual systemd services with a single orchestrator. The supervisor reads JSON mode configurations and manages the entire process chain — from rigctld and Direwolf to end-user applications like YAAC, Pat (Winlink), or JS8Call.

System Architecture at a Glance

Component Location Purpose
et-supervisor bin/et-supervisor Main daemon — process lifecycle, IPC, health checks
et-supervisor-ctl bin/et-supervisor-ctl CLI client — start/stop modes, query status

Persistence bin/et-persistence/ Save/restore configs, documents, WiFi, BT to USB

Supported Software

Winlink

Email over radio via Pat client. Supports VARA HF, VARA FM, ARDOP, and VHF/UHF Packet.

APRS

Automatic Packet Reporting System via YAAC client and Direwolf TNC. Supports digipeater mode.

JS8Call

Keyboard-to-keyboard HF messaging based on FT8 modulation. Store-and-forward capable.

WSJT-X (FT8/FT4)

Weak signal digital modes for HF propagation testing and contacts.

VarAC

VARA Chat — keyboard-to-keyboard chat over VARA HF modem. Runs via Wine.

fldigi

Multi-mode digital modem: PSK31, RTTY, Olivia, MFSK, and many more modes.

Chattervox

AX.25 keyboard chat with cryptographic identity verification (secp256k1 signing).

LinBPQ

Packet BBS server with AX.25 networking. Runs alongside Direwolf TNC.

Quick Start

  1. Download or Build — Build the ISO with setup-emcomm-iso.sh (see Building the ISO)
  2. Write to USB — Use dd or Balena Etcher to flash the ISO to a USB drive
  3. Boot & Configure — The First Boot Wizard guides you through callsign, radio, and data setup
  4. Start a Mode — Click a mode button on the dashboard or use et-supervisor-ctl start <mode>
