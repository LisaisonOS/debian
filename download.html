How to install

1
Download the ISO file using the button above.
2
Verify the SHA256 checksum to confirm the file was not corrupted during download.
3
Write to a USB drive using balenaEtcher (recommended for beginners) or Ventoy (multi-boot capable).
4
Boot from USB — press F12, F2, or ESC at startup to access your boot menu. On Mac, hold Option.
5
Select your language (EN/FR) at first boot, configure your callsign and radio, and you are on the air.
6
Optional: Install permanently to your hard drive using Calamares (installer icon on the desktop).

USB boot creator tools

Windows

  • Rufus — fast, reliable, open-source
  • balenaEtcher — simple 3-step
  • Ventoy — multi-boot capable

macOS

  • balenaEtcher — best for Mac
  • UNetbootin — cross-platform
  • dd (Terminal) — built-in

Linux

  • balenaEtcher — AppImage
  • Ventoy — copy ISOs to USB
  • GNOME Disks — pre-installed
  • dd — classic, fast
